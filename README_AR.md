# NT2 Lezen B1 — النماذج الرسمية 2023–2025

موقع تدريب غير رسمي مخصص حصريًا لنماذج Lezen I للأعوام 2023 و2024 و2025.

## المحتوى

- 3 نماذج رسمية كاملة.
- 18 نصًا رسميًا.
- 105 أسئلة متعددة الخيارات: 35 سؤالًا لكل سنة.
- 50 مثالًا موثقًا في مكتبة إعادة الصياغة.
- 59 علاقة لغوية في لعبة المعنى.
- ملفات النصوص ومفاتيح الإجابة للأعوام 2023 و2024 و2025 فقط.

## استراتيجية القراءة داخل الموقع

ابدأ بالسؤال والاختيارات، ثم حدد كلمة البحث واقرأ الجزء المرتبط بالسؤال. لا تحاول قراءة النص كلمةً كلمة أو ترجمة جميع المفردات.

## التشغيل

```bash
npm ci --no-audit --no-fund
npm run check
npm run build
```
